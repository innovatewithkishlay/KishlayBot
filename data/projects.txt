ProcessPilot / Task Manager  
ProcessPilot is a web-based tool for monitoring and managing system processes. Features real-time CPU and memory tracking, process filtering, termination, and an AI-powered chatbot for system queries. The app includes a dynamic CPU usage graph and displays key system details in a fully responsive design.  
Tech stack: React.js, Tailwind CSS, Framer Motion, Python, PsUtil, Flask  
Source: https://github.com/innovatewithkishlay/ProcessPilot
